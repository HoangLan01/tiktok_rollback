# Project Rules & Style Guide
1. **Visual Style:** - Theme: Deep Dark Mode (Background: #121212).
   - Typography: Use Sans-serif (Inter or Montserrat). High contrast for key metrics.
   - Colors: Primary Green (#22C55E), Blue (#3B82F6), Red (#EF4444), Orange (#F59E0B).
2. **Component Architecture:**
   - Atomic Design: Break UI into atoms (Badges, Buttons), molecules (StatCards), and organisms (Top5List, BrandChart).
3. **Responsiveness:** Mobile-first approach. Ensure the "Year in Review" look translates well to desktop.
4. **Data Handling:** Use mock data initially, but structure APIs to be ready for TikTok Research API or Scraper integration.
